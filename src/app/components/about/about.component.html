🔥 Open-Source Initiatives

I have a strong commitment to the open-source community. My most notable contribution is Laradock.io, a full PHP development environment for Docker. It has been used by thousands of developers to simplify their workflow.

In addition, I created Apiato, a flawless framework for building scalable APIs on top of Laravel, designed to speed up development while maintaining code quality.

My work also includes Porto, a software architectural pattern that helps developers organize their code in a highly modular way.

⚡ Cloud & AI Specialization

My focus lies at the intersection of Cloud Infrastructure and Artificial Intelligence.

I specialize in building robust RAG (Retrieval-Augmented Generation) systems that allow AI models to interact with private data securely and efficiently.

I also have deep experience in MLOps, deploying and monitoring machine learning models using Kubernetes and Kubeflow, ensuring they perform reliably in production environments.
